# Stoic Content Dataset

This project uses `assets/data/stoic_content.json` as the source for `lib/features/stoic/*`.

## Structure

Each entry has:

- `id`
- `title`
- `content`
- `directive`
- `category`
- `intensity`
- `icon`

`content` supports tagged lines used by the UI:

- `[VISUAL]`
- `[GRAMMAR]`
- `[STRATEGY]`
- `[GRAPH]`
- `[DATA]`

## Regeneration

The dataset is generated by:

```bash
python3 tools/gen_stoic_content.py
```

Generator behavior:

- Curates source themes from `MD/stoic.rtf` (validated keyword sections).
- Produces `65` lessons per Stoic category.
- Writes output to `assets/data/stoic_content.json`.
